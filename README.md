# Netflix Data Analysis

This project analyzes Netflix movies and TV shows to discover trends in content production, ratings, and genres.

## Tools Used
Python
Pandas
Matplotlib
Google Colab
Power BI

## Analysis Performed
Data Cleaning
Exploratory Data Analysis
Data Visualization
Dashboard Creation

## Key Insights
• Netflix has more movies than TV shows
• Content production increased after 2015
• United States produces most content

## Dashboard
Interactive dashboard created using Power BI.
