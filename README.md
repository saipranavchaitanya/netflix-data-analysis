# Netflix Data Analysis


## Project Overview

This project analyzes a Netflix movies and TV shows dataset to uncover trends in content production, ratings, popularity, and global distribution.
The goal is to explore the dataset, clean the data, perform exploratory data analysis (EDA), and create visualizations to understand how Netflix content varies across years, countries, and genres.

The analysis also demonstrates practical data analyst skills including data cleaning, visualization, and dashboard creation.

---

## Objectives

The main objectives of this project are:

• Understand the structure of the Netflix dataset
• Clean and prepare the dataset for analysis
• Explore trends in Netflix content over time
• Identify the most popular genres and countries producing content
• Analyze ratings and popularity of movies and TV shows
• Build a visual dashboard to communicate insights

---

## Dataset

The dataset used in this project contains information about Netflix titles including movies and TV shows.

Dataset Source: Kaggle

The dataset includes the following key columns:

* Title
* Type (Movie or TV Show)
* Release Year
* Country
* Genres
* Language
* Rating
* Popularity
* Vote Average
* Vote Count
* Budget
* Revenue

## Data Cleaning Process

Before analysis, several preprocessing steps were performed:

• Checked dataset structure and data types
• Handled missing values in important columns
• Removed or replaced null values
• Converted columns to correct data types
• Prepared categorical variables for analysis

Cleaning the data ensures accurate analysis and reliable insights.

## Exploratory Data Analysis (EDA)

The following analysis was performed:

### 1. Movies vs TV Shows Distribution

Analyzed the distribution of movies and TV shows available on Netflix.

### 2. Content Production by Year

Examined how the number of Netflix titles released has changed over time.

### 3. Top Countries Producing Content

Identified which countries produce the most Netflix content.

### 4. Genre Analysis

Explored the most common genres available on Netflix.

### 5. Ratings Distribution

Analyzed how content is rated across different categories.

### 6. Popular Titles

Examined which titles have the highest popularity scores and ratings.

## Dashboard

An interactive dashboard was created using Power BI to visualize insights from the dataset.

The dashboard includes:

• Key performance indicators (KPIs)
• Movies vs TV Shows distribution
• Content released per year
• Top producing countries
• Genre distribution
• Ratings overview
• Movie popularity table

The dashboard allows users to filter results by year, country, language, and content type.

---

## Key Insights

Some important insights discovered during the analysis:

• Netflix has significantly more movies than TV shows
• Content production increased rapidly after 2015
• The United States produces the highest number of titles
• Drama and international movies are among the most common genres
• Highly rated content tends to have higher popularity scores
